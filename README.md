# FP-A-Budget-Variance-Analysis
Financial analysis dashboard comparing budget vs actual spending to identify variances and cost control opportunities.
### Tools Used
Microsoft Excel, Pivot Tables, Variance Analysis

### Analysis Performed
- Compared Budget vs Actual spend
- Calculated variance and variance %
- Identified overspending departments
- Created dashboard visualizations

### Key Insights
• Marketing exceeded budget due to campaign spend  
• IT overspend driven by software & infrastructure upgrades  
• HR maintained cost efficiency within budget  

### Business Recommendations
Implement monthly variance monitoring and approval controls to prevent overspending and improve cost discipline.
